# Contributing to Keti OS
Keti OS is an x86 operating system built from scratch.
Built by Aadithya Vasudevan (Aadithya-19).

## How to contribute
- Fork the repo
- Pick an open issue or suggest a feature
- Submit a pull request with a clear description

## Ground rules
- All code must be understood by the author (primarily you) before merging
- No blatant copy-paste - explain what your code does
- Keep it fun
- Try to give a shoutout to the resources you used and we can expand the resource board available.

## Open Works
- You can add your ideas here that is beyond your scope and maybe someone who knows how can add it.
